## 🇷🇴 Despre acest repository

Acest repository documentează proiectele mele practice de electronică analogică și digitală. Fiecare proiect parcurge întreg procesul: de la simulare în **LTspice** și **Logisim Evolution**, până la implementarea fizică și măsurători reale.

Scopul este să demonstrez înțelegerea principiilor de electronică prin proiecte documentate.

### 🎯 Ce conține fiecare proiect
- 📐 **Schemă** — circuit proiectat în LTspice sau Logisim Evolution
- 🧮 **Calcule** — dimensionarea componentelor, verificare teoretică
- 📊 **Simulare** — rezultate din LTspice / tabele de adevăr din Logisim
- 🔌 **Implementare fizică** — poze cu circuitul pe breadboard
- 📏 **Măsurători reale** — comparație valori teoretice vs. măsurate
- 📚 **Ce am învățat** — concluzii și lecții extrase

## 🇬🇧 About this repository

This repository documents my hands-on analog and digital electronics projects. Every project follows the full engineering workflow: from simulation in **LTspice** and **Logisim Evolution**, to physical implementation and real-world measurements.

The goal is to demonstrate understanding of electronics principles through documented projects.

### 🎯 What each project contains
- 📐 **Schematic** — circuit designed in LTspice or Logisim Evolution
- 🧮 **Calculations** — component sizing, theoretical verification
- 📊 **Simulation** — LTspice results / Logisim truth tables
- 🔌 **Physical build** — photos of the breadboard circuit
- 📏 **Real measurements** — theoretical vs. measured values comparison
- 📚 **Lessons learned** — conclusions and takeaways
